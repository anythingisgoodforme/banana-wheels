Banana Wheels Project Structure
==============================

banana-wheels/
|-- public/
|   |-- index.html      # Main page and HUD markup
|   |-- styles.css      # Layout and visual styling
|   |-- game.js         # Current first-person game runtime
|   `-- assets/         # Optional images/audio if added later
|-- src/                # Optional shared modules and examples
|-- tests/              # Jest tests for reusable code
|-- README.md           # Project overview and scripts
|-- QUICKSTART.md       # Fast setup guide
|-- CONTRIBUTING.md     # Development workflow
|-- package.json        # Scripts and dependencies
`-- jest.config.js      # Test configuration

Primary workflow
----------------
1. Run `npm install` once.
2. Run `npm run dev` to serve the game on port 8000.
3. Edit `public/game.js` for gameplay changes.
4. Run `npm run lint`, `npm run format:check`, and `npm test` before pushing.
